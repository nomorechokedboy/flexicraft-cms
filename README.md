# Requisite

`Go` 1.18+

`Node` 16 LTS

# Getting started

- How to run go server

```sh
git clone https://github.com/nomorechokedboy/go-oauth2.git
cd go-oauth2
go build .
./api
```

- How to run frontend

```sh
cd frontend
pnpm i # yarn or npm i
pnpm dev
```

# Features

- Sign up
- Sign in
- Oauth (phải chạy project frontend)
